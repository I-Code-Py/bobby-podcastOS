# Bobby San Podcast — Clip Payout Tool

Outil de calcul automatique de rémunération des clippeurs (1€/1000 vues), avec
détection des republications d'un même clip sur TikTok, Instagram Reels et
YouTube Shorts (fingerprint vidéo + audio) pour éviter les doublons de paiement.
